# B 站视频脚本（约 90 秒）

## 0–8 秒：钩子

屏幕录制普通 Agent 回答，再切到启用后的回答。

旁白：`我给 Agent 装了一个火影玩家语言系统。现在它不止会写代码，它还会给报错开追悼会。`

## 8–25 秒：产品可信度

展示仓库结构、`./install.sh --target codex` 与 Agent 识别 Skill 的画面。

旁白：`这不是语录复制器。它会从你的问题里自己抽词、造句，而且不玩火影也能听懂。`

## 25–60 秒：三段实测

1. Bug 排查：缓存/依赖问题。
2. 离谱聊天记录点评。
3. 普通概念解释。

每段先放一行用户问题，再展示有完整解决方案的风格回答。保留命令和代码的原样可读性。

## 60–75 秒：边界

展示严肃问题和危险请求：前者清晰克制，后者明确拒绝。字幕：`整活归整活，答案不能掉线。`

## 75–90 秒：彩蛋与行动

用火影手游提问收尾，让 Agent 自然切入少量决斗场表达。展示 GitHub 链接与安装命令。

标题建议：`我给Agent装上火影玩家语言系统后，它连不玩火影的人都能整笑 〖B站AI创造公开赛〗`
